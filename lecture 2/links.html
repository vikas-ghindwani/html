Links

Used as 'a' tag.

External Links

External links are those types of links where u sending a user for your website to someone else's website.

Now, there r 2 types of External links:

1. {target="_self"} means that u are loading someone else's website on the same page, that will close ur page and so no trafficking on ur page. therefore not good.
Ex. youtube.com

2. {target="_blank"} means that u r opening someone else's website on a new page. that is good as user is still on your website.
Ex. youtube.com


Internal Links

Internal links are those types of links where u sending a user from one of your page to YOUR other page.

Now, there r 2 types of Internal links:

  • Sending a user to ur page which is stored in same folder.
    can be done in 2 ways:

    1. absolute link: Means tht u r starting to come from the main folder to inner ones.

      written as:
      < a href="/html/lecture 2/second.html" target="_blank">lec2 second link < / a >

      Ex. lec2 second link
      ['/' for absolute link]

    2. relative link: Means tht u just have to write the name of the file. it will work the same as long as that file is in the same folder as your starting file.

      written as:
      < a href="second.html" target="_blank"> lec2 second link < / a >

      Ex. lec2 second link

  • Sending a user to ur page which is stored in other folder.
    can be done in 2 ways:

    1. absolute link: Means tht u r starting to come from the main folder to inner ones.

      written as:
      < a href="/html/lecture1/tags.html" target="_blank">lec1 tags < / a >

      Ex. lec1 tags
      [ '/' for absolute link]

    2. relative link: Means tht u r going to another folder -> file relative to current file
      means that the path will be: coming out of current file -> folder then go to the needed folder -> file.

      written as:
      < a href="../lecture1/tags.html">lec1 tags < / a >

      Ex. lec1 tags
      ['../' for relative link]
