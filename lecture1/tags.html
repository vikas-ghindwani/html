These are all types of tags:

1. < h(1-6) >: Used for writing headers. can be used mulyiple times BUT should preferably used 1 time.
they r of 6 types h1 to h6.
h1 has biggest font and h6 has the smallest font

2. < p >: Used for writing paragraphs, can be used multiple times.
it ues full block, means that everything else after 'p' tag will continue from next line even though how small a paragraph is continuing on a line.

3. < pre >: can be used similarly as paragraph tag BUT preferably not to be used much.
it shows the exact size and font of the text which is written in pre tag.

4. < hr >: Used to draw a horizontal line.

5. < lorem >: this tag is used to generate random words.
type loremX to generate X no. of random words.

6. < br >: Used to go to next line.

7. < b >: It is a non semantic tag.
used to highlight a word or a sentence in bold format for highlighting something to user.

8. < i >: It is a non semantic tag.
used to highlight a word or a sentence in italic format for highlighting something to user.

9. < strong >: It is a semantic tag.
looks just like 'b' tag, but used to highlight a specific word or sentence to user, but also used for showing the link in search results if a user searches for that word or sentence.

10. < em >: It is a semantic tag.
looks just like 'i' tag, but used to highlight a specific word or sentence to user, but also used for showing the link in search results if a user searches for that word or sentence.

11. < span >: Works just like a paragraph tag, but for 2-3 words.
but it doesnt take full block as paragraph tag.
